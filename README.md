# Notification



## Environment setup

### 1. Prerequisites

* Python v3.6
* Django v2.1.5
* Virtualenv v16.4.0

### 2. Clone project


```sh
$   git clone https://github.com/Ayasser/Notification.git
```
    
### 3. Create Virtualenv

```sh
$   virtualenv env
 ```

### 4. Activate Virtualenv

```sh
$ source env/bin/activate
```

### 5. Install required packages inside virtualenv

```sh
$(env)  pip install -r requirements.txt
```

### 6. Verify

```sh
$(env)  manage.py runserver
```
